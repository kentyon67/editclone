# EditClone 仕様書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | EditClone |
| リポジトリ | https://github.com/kentyon67/editclone |
| 言語 | Python |
| 最終目標 | Final Cut Pro向けAI編集アシスタント |
| コア価値 | **編集スタイルの再現** |

EditCloneは、ユーザーが動画をアップロードするだけで、AIが編集スタイルを分析し、Final Cut Pro用のプロジェクトファイル（FCPXML）を生成するツールです。
参考動画のスタイルを学習させる「モデル動画機能」と、編集前後の差分から編集ルールを抽出する「ペア分析機能」が最大の差別化ポイントです。

## 3. 機能仕様

### 3.1 MVP（初期リリース）で実装する機能

| # | 機能 | 説明 |
|---|------|------|
| 1 | ヘルスチェック | `GET /health` でサーバー生存確認 |
| 2 | 動画アップロード | 動画ファイルをサーバーに保存 |
| 3 | 動画基本情報取得 | ファイル名・サイズ・動画長さを返す |
| 4 | Whisper文字起こし | 音声をテキスト化し、タイムスタンプ付きで返す |
| 5 | 無音検出 | 無音区間のタイムスタンプ一覧を返す |
| 6 | カット提案 | 無音区間・セリフ区切りからカット候補を提案 |
| 7 | FCPXML生成 | カット提案をFinal Cut Proで読み込めるFCPXMLとして出力 |

## 4. APIエンドポイント仕様（MVP）

### GET /health

サーバーの生存確認。

**レスポンス:**
```json
{ "status": "ok" }
```

### POST /videos/upload

動画ファイルをアップロードする。

**リクエスト:** `multipart/form-data`
- `file`: 動画ファイル（mp4, mov 等）

**レスポンス:**
```json
{
  "video_id": "uuid",
  "filename": "example.mp4",
  "size_bytes": 104857600,
  "duration_seconds": 300.5
}
```

### GET /videos/{video_id}/info

動画の基本情報を取得する。

**レスポンス:**
```json
{
  "video_id": "uuid",
  "filename": "example.mp4",
  "size_bytes": 104857600,
  "duration_seconds": 300.5,
  "width": 1920,
  "height": 1080,
  "fps": 30.0
}
```

### POST /videos/{video_id}/transcribe

Whisperで文字起こしを実行する。

**レスポンス:**
```json
{
  "video_id": "uuid",
  "segments": [
    { "start": 0.0, "end": 2.5, "text": "こんにちは" }
  ]
}
```

### POST /videos/{video_id}/detect-silence

無音区間を検出する。

**レスポンス:**
```json
{
  "video_id": "uuid",
  "silence_segments": [
    { "start": 5.0, "end": 6.2 }
  ]
}
```

### POST /videos/{video_id}/suggest-cuts

カット候補を提案する。

**レスポンス:**
```json
{
  "video_id": "uuid",
  "cut_suggestions": [
    { "start": 5.0, "end": 6.2, "reason": "silence" }
  ]
}
```

### POST /videos/{video_id}/export-fcpxml

FCPXML形式でエクスポートする。

**レスポンス:** `application/xml`（FCPXMLファイルのダウンロード）

| 形式 | 内容 |
|------|------|
| 動画ファイル | mp4, mov, avi 等（ffmpegがサポートする形式） |
| 最大ファイルサイズ | MVP時点では未定（ローカル動作前提） |

### 出力

| 形式 | 内容 |
|------|------|
| JSON | APIレスポンス（文字起こし・カット提案等） |
| FCPXML | Final Cut Pro用プロジェクトファイル |
